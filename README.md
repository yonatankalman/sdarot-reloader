# Sdarot Reloader

Chrome extension to reload sdarot.work when failing to find server until it is successful.

## To enable:

* Visit chrome://extensions (in the chrome browser)
* Toggle "Developer mode" on
* Click "load unpacked"
* Select the sdarot-reloader directory.
